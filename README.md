🚀 Student Doubt Solver

A web application built using ASP.NET Core MVC where students can ask academic questions and receive answers from other users in a collaborative learning environment.

⚙️ Installation

📥 Clone the repository

```bash
git clone https://github.com/Aarnavyas1304/StudentDoubtSolver.git
```

▶️ Open the project in Visual Studio.

📦 Restore NuGet packages.

🗄️ Configure the SQL Server connection string in `appsettings.json`.

🚀 Run the application.
